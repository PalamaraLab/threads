Threads
=======

Threads infers ancestral recombination graphs (ARGs) from phased genotype data
and stores them as compact *threading instructions*. This implicit
representation supports downstream analyses---allele frequency spectrum,
association testing, mutation generation, genotype-matrix multiplication---without
materializing an explicit ARG.
